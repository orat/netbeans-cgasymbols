# netbeans-cgasymbols
Adds a context menu in the netbeans editor for easy inserting cga specific symbols.

This is a Netbeans-IDE plugin, which adds a submenu into the editors context-menu to insert CGA-specific symbols and operators into the Editor needed by the [DSL](https://github.com/MobMonRob/DSL4GeometricAlgebra/).

To use it build the Netbeans project and reload it into the running IDE.
